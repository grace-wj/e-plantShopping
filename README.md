# e-plantShopping

**Paradise Nursery** – Front-end shopping application for browsing houseplants and managing a shopping cart.

This project is the **e-plantShopping** repository.

## Overview

- **Landing page** – Welcome message, company intro, and Get Started button
- **Product listing** – Houseplants by category with Add to Cart
- **Shopping cart** – View items, adjust quantities, remove items, and see totals

## Tech Stack

- React (Vite)
- Redux Toolkit & react-redux

## Run locally

```bash
npm install
npm run dev
```
